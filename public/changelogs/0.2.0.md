[detection.studio](https://detection.studio/) now has included pipeline support for pySigma backends - such as ECS Windows, Splunk Windows and more. 

## Migration to pySigma 1.0

With the migration to pySigma 1.0, https://detection.studio/ now supports the latest features and improvements from the pySigma project. This includes enhanced rule parsing, better performance, and support for new detection techniques.
